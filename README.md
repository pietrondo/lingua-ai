# Language AI

Collection of languages, tools and experiments for AI-native code generation and reasoning.

## Projects

### LINGUA

AI-optimized programming language with semantically rich syntax designed for LLM code generation and reasoning.

**Location:** `lingua/`

```bash
# Parse a .lingua file
lingua parse saluto.lingua

# Generate Python
lingua python saluto.lingua

# Generate JSON
lingua json saluto.lingua

# Generate IR (knowledge graph)
lingua ir saluto.lingua
```

**Example (`hello.lingua`):**
```lingua
§
╔═══╗ CONCEPT hello
│ TYPE operazione
│ INPUT []
│ OUTPUT stringa
│ REASON "Generates a greeting in English"
│ IMPL "Hello, world!"
╚═══╝
§
```

**Compiles to:**
```python
def hello() -> str:
    """Generates a greeting in English"""
    return "Hello, world!"
```

### Nuitka Compilation

Python programs generated by LINGUA can be compiled to native binaries via [Nuitka](https://nuitka.net/):

```bash
pip install nuitka
python -m nuitka --onefile hello.py   # requires Dependency Walker on Windows
python -m nuitka hello.py             # creates hello.exe
```
